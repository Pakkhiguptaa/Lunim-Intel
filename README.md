# Lunim Intel Dashboard

A premium, real-time Competitor Intelligence Dashboard.

## Features
- **Live Press Signals**: Fetches the latest news and press releases using the Tavily Search API.
- **AI Summarization**: Generates concise, one-line intelligence summaries using GitHub Models (GPT-4o).
- **Modern UI**: Sleek, deep-blue aesthetic designed for high-end intelligence monitoring.
- **Configurable**: Easily connect your own API keys to get live updates.

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.
3. Enter your **Tavily API Key** and **GitHub Personal Access Token** (GitHub Models) in the "Live Connector" section.
4. Click "Run Intelligent Search" to fetch live data.

## Technologies
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- [Tavily AI Search](https://tavily.com/)
- [GitHub Models](https://github.com/marketplace/models) (GPT-4o)
